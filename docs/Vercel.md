# Vercel (버셀)

> "프론트엔드 개발자들을 위한 가장 쉽고 빠른 배포 플랫폼"

Next.js의 제작사인 Vercel에서 운영하는 클라우드 플랫폼이다. 프론트엔드 프레임워크와 정적 사이트에 최적화되어 있으며, Git 호스팅 서비스(GitHub, GitLab 등)와 연동하여 코드 푸시만으로 자동 배포되는 경험을 제공한다.

## 왜 쓰는가? (강점)

- **Zero Config**: 설정 파일 없이도 소스 코드만 있으면 알아서 빌드하고 배포한다.
- **Preview Deployments**: 풀 리퀘스트(PR)를 올릴 때마다 고유한 미리보기 URL을 생성해준다. 협업할 때 최고다.
- **Edge Network**: 전 세계 어디서든 빠른 로딩 속도를 보장하기 위해 에지 컴퓨팅 기술을 적극적으로 활용한다.
- **Serverless Functions**: 별도의 백엔드 서버 없이도 API 엔드포인트를 간단하게 만들 수 있다.

## 연관 기술

- **[[Next.js]]**: Vercel이 직접 만든 프레임워크인 만큼, 최고의 궁합과 성능을 보여준다.
- **Serverless**: 인프라 관리 없이 코드만 실행되는 환경을 지향한다.
- **Edge Runtime**: 전 세계 분산된 서버에서 더 빠르게 코드를 실행하는 런타임.

## 작동 방식 (Workflow)

1. **연동**: GitHub 등 VCS 계정을 Vercel에 연동한다.
2. **Push**: 코드를 메인 브랜치에 `git push` 한다.
3. **Build & Deploy**: Vercel이 변경 사항을 감지하고 자동으로 빌드 후 서버에 올린다.
4. **Finish**: 수 초 내에 라이브 URL이 생성된다.

## 특징 및 장점

- **SSL 자동 적용**: 별도 설정 없이 무료로 HTTPS가 적용된다.
- **Custom Domain**: 개인 도메인을 연결하는 과정이 매우 직관적이고 빠르다.
- **Analytics**: 사이트 성능(Core Web Vitals)을 실시간으로 분석해준다.
- **무료 플랜 (Hobby)**: 개인 프로젝트용으로 쓰기에 충분히 관대한 무료 요금제를 제공한다.

## 기타

- 원래 이름은 **ZEIT**였으나, Vercel로 사명을 변경했다.
- [[Tailwind CSS]] 제작사가 어려움에 처했을 때 긴급 후원에 나서는 등, 프론트엔드 생태계 전체에 큰 영향력을 행사하고 있다.

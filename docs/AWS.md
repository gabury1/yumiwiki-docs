# AWS (Amazon Web Services)

> 클라우드 컴퓨팅의 시작이자 표준

아마존이 제공하는 세계 1위 클라우드 컴퓨팅 플랫폼. 서버, 스토리지, 데이터베이스, 네트워킹, AI 등 200개가 넘는 서비스를 제공한다. "인프라를 전기나 수도처럼 쓴다"는 개념을 대중화시켰다.

## 주요 서비스 (Core Services)

### 컴퓨팅
- **EC2 (Elastic Compute Cloud)**: 가상 서버(VM). 컴퓨터 한 대 빌리는 것.
- **Lambda**: [[Serverless]] 컴퓨팅. 서버 없이 코드만 올리면 실행된다. 이벤트 기반.

### 스토리지
- **S3 (Simple Storage Service)**: 무한히 확장 가능한 객체 스토리지. 파일, 이미지, 백업 저장소. 정적 웹사이트 호스팅도 가능.

### 데이터베이스
- **RDS**: 관계형 데이터베이스([[MySQL]], [[PostgreSQL]] 등) 관리형 서비스. 백업, 패치를 알아서 해준다.
- **DynamoDB**: 초고속 NoSQL 키-값 데이터베이스.

### 네트워킹
- **VPC**: 가상 사설 네트워크. 클라우드 안에 나만의 격리된 네트워크망 구축.
- **Route 53**: DNS(도메인) 서비스.

## 특징

**On-Demand**: 필요할 때 생성하고, 쓴 만큼만 돈을 낸다(Pay-as-you-go). 서버를 미리 사놓을 필요가 없다.

**가용성**: 전 세계에 리전(Region)과 가용 영역(AZ)이 퍼져 있다. 하나가 불타도 다른 곳에서 서비스가 돌아가게 설계할 수 있다([[고가용성]], [[재해 복구]]).

**IaC 연동**: [[Terraform]]이나 AWS CDK를 사용해 인프라를 코드로 관리하고 찍어낼 수 있다.

## 생태계

경쟁자로 Microsoft Azure, Google Cloud([[GCP]])가 있지만, AWS가 점유율과 성숙도 면에서 압도적인 1위다. 대부분의 스타트업과 엔터프라이즈가 AWS를 쓴다.

## 기타

Jeff Bezos가 "내부 시스템을 외부에도 팔 수 있게 API로 잘 정리해라"라고 지시한 것(비 피자 두 판의 법칙 등)이 시작이었다는 이야기가 유명하다.

너무 서비스가 많아서 "자격증 장사"라는 말도 듣지만, Solutions Architect 같은 자격증은 업계에서 꽤 인정받는다.
